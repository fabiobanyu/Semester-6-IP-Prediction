# 🎓 IP & IPK Predictor (Semester 6)

Aplikasi kalkulator dan simulasi IPK interaktif yang dirancang khusus untuk memprediksi Indeks Prestasi (IP) dan Indeks Prestasi Kumulatif (IPK) di Semester 6 berdasarkan bobot penilaian tiap komponen mata kuliah. 

🚀 **[Live Demo: Coba Aplikasinya Sekarang!](https://fabiobanyu.github.io/Semester-6-IP-Prediction/)**

## ✨ Fitur Utama
- **Kalkulator Akurat:** Hitung skor akhir setiap mata kuliah (Tugas, Kuis, UTS, UAS, Praktikum) sesuai persentase bobot penilaian tanpa hitung manual.
- **Konfigurasi SKS Dinamis:** Pilih mata kuliah apa saja yang Anda ambil di semester ini (Sistem akan mengkalkulasi SKS secara real-time).
- **Simulasi Target IPK:** Masukkan IPK Anda saat ini dan target kelulusan Anda. Sistem akan membeberkan berapa target minimal nilai Semester 6 yang perlu dicapai.
- **Modern & Cepat:** Berjalan 100% secara instan di *browser* (tanpa *loading*, tanpa server), dan dibalut dengan UI Dark Mode eksklusif nan elegan.

## 🛠️ Dibangun Menggunakan
- HTML5, CSS3, dan Vanilla JavaScript.
- Tidak membutuhkan instalasi *backend* atau *environment* apa pun.

## 📖 Cara Penggunaan
1. Buka [tautan aplikasi](https://fabiobanyu.github.io/Semester-6-IP-Prediction/).
2. Pada tab **Mata Kuliah**, centang mata kuliah yang Anda ikuti semester ini.
3. Gunakan tab **Kalkulator Nilai** untuk memprediksi angka akhir berdasarkan nilai masing-masing tugas dan ujian dari setiap dosen.
4. Pada tab **Simulasi IPK**, atur nilai huruf (A, AB, B, dll) untuk tiap mata kuliah guna menyimulasikan dampak eksaknya terhadap IPK Anda!

## 💡 Konversi Skala Nilai
Aplikasi ini secara *default* menggunakan standar konversi berikut (dapat disesuaikan di `script.js` jika diperlukan):
- **A**  : 75 - 100
- **AB** : 70 - 74.99
- **B**  : 65 - 69.99
- **BC** : 60 - 64.99
- **C**  : 50 - 59.99
- **D**  : 40 - 49.99
- **E**  : 0 - 39.99
